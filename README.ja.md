# event_fukui_data

このリポジトリには、福井県内のイベント情報を収集・整理したデータが含まれています。主に「ふくいイベントNet」のサイトから情報を取得し、CSV形式でまとめています。

## データ・API
このプロジェクトでは、以下のデータソースを使用しています:

- [ふくいイベントNet](https://www.fuku-e.com/070_event/)

取得したイベント情報は、以下のCSVファイルにまとめられています:

- `data/event_fukui.csv`

このCSVファイルには、イベントのタイトル、開催日時、開催場所、料金、連絡先情報などが含まれています。
